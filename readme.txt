== Description ==

Implementace sms brány SMSVio do WooCommerce 

SMSVio pro WooCommerce vám umožní posílat SMS oznámení automaticky pro objednávky umístěné v WooCommerce. 
Po vytvoření objednávky automaticky kupující održí SMS zprávu, stejně tak může být formou SMS informován administrátor eshopu. Všechny SMS zprávy přizpůsobit a upravit z administrace. 
Jedná se o rozšíření WooCommerce, které obsahuje oblíbenou SMS bránu SMSVio, kdy využíváte vašeho stávajícího mobilního tarifu.

Funkce

Při vytvoření a jakékoliv aktualizaci stavu objednávky odešle zákazníkovi SMS oznámení
Plně přizpůsobitelný text SMS pro každou stavovou zprávu.
Podrobnosti o odeslání objednávky (číslo objednávky, stav objednávky, název objednávkového obchodu a množství objednávek a mnoho dalších podporovaných zkratek) v SMS textu.

Popis brány SMSVio

Brána SMSVio Vám umožní zasílat posílat hromadné SMS všem zákazníkům vašeho eshopu, i pomocí xls souboru s nastavením data a času odeslání
Pronajímáte si SMS bránu jako takovou, neplatíte tedy poplatky navíc za odesílání SMS, vyjma ceny vyplývající z vašeho stávajícího mobilního tarifu.
Odhlášení z odběru zpráv pmocí odpovědní SMS ve tvaru Odhlasit

== Installation ==

Instalujte a aktivujte plugin SMSVio pro WooCommerce. Přejděte v levém menu na nabídku WooCommerce - SMSVio pro WooCommerce
Pro správnou instalaci budete potřebovat API klíč, který najdete v administraci vašeho uživatelského účtu SMSVio pod záložkou API, jako číslo zařízení zpravidla zadejte 0.

== Frequently Asked Questions ==

= Jak mám plugin aktivovat ? =
K použití pluginu budete potřebovat účet [SMSVio](https://smsvio.cz). Můžete se zaregistrovat [zde](https://admin.smsvio.cz/register.php). 
Následně emailem obdržíte vaše přihlašovací údaje. Poté se stačí přihlásit do webové aplikace smsvio a přejít na stránku API. Zde najdete klíč API. Tento API klíč vložíte do nastavení pluginu SMSVio, tím začne být plugin plně funkční.
